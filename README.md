# vue-cli3-h5-templet

## 主要功能

```angular2html
1. 常用目录别名
2. mint-ui/Rem适配
3. scss支持、_mixin.scss、_variables.scss
4. 页面切换动画+keepAlive
5. 页面标题
6. 自动注册：自动注册路由表/自动注册Vuex
7. axios封装、api管理
8. 用户鉴权
9. vuex-loading
10. 生产环境优化
```

yarn install
